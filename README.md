# Java Spring Boot CRUD Usuários

Desenvolvimento em Java com Spring Boot para gerenciamento de usuários, permitindo realizar operações de cadastro, consulta, atualização e exclusão (CRUD).

O projeto segue boas práticas de desenvolvimento, organização em camadas e comunicação via HTTP, sendo ideal para estudos e demonstração de conhecimentos em backend com Java.

## Tecnologias
- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- Banco de dados (MySQL ou H2)
- Maven
